# Future Roadmap

## Short-term improvements

- Add actual live cluster discovery from kubeconfig.
- Add cluster creation and Kind bootstrap support.
- Add node health and pod-level detail panels.
- Improve 3D rendering with GPU instancing for thousands of pods.
- Add a scatter/guided layout to render full-scale galaxy views.

## Mid-term features

- Add terminal access and live logs.
- Add manifest editor with diff preview.
- Add filters and search for pods, services, and namespaces.
- Add cluster operations: scale, drain, cordon, delete.
- Add service and network topology visualizations.

## Long-term vision

- Add audit trail, incident replay, and history timeline.
- Add multi-team workspaces and role-aware actions.
- Add CI/CD pipeline triggers and deployment dashboards.
- Add anomaly detection and proactive remediation suggestions.
- Add immersive VR/3D controls for infrastructure exploration.
